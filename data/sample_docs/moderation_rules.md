# Правила модерации объявлений (регламент)

## 1. Цель модерации

- обеспечить безопасность пользователей;
    
- снизить мошенничество;
    
- поддерживать качество каталога: корректные категории, честное описание, понятные фото.
    

## 2. Что проверяется в каждом объявлении

1. **Категория и атрибуты**: соответствие товара/услуги выбранной категории.
    
2. **Название**: без капса, без “супер/топ/лучший”, без контактных данных.
    
3. **Описание**: фактическое состояние, комплектность, дефекты, ограничения.
    
4. **Фото**: реальные, без водяных знаков “других площадок”, без коллажей с ценниками “из интернета”.
    
5. **Цена**: реалистичность и отсутствие скрытых платежей (“цена от…”, “пишите — договоримся” допускается только в услугах).
    
6. **Контактные данные**: нельзя размещать телефон/мессенджеры в тексте/картинке.
    

## 3. Частые причины отклонения

- Несоответствие категории (“смартфон” размещён как “аксессуары”).
    
- Фото не товара: скриншоты, картинки из каталога, размытые изображения.
    
- Запрещённые товары/услуги (см. политику запретов).
    
- Указание “оплата только переводом”, “без безопасной сделки” в описании.
    
- Дублирование: одно и то же объявление размещено 3+ раза с минимальными изменениями.
    
- Манипуляция поиском: набор ключевых слов, не относящихся к товару.
    

## 4. Редактирование и повторная подача

- После отклонения пользователь видит причину и список требований.
    
- Повторная подача допускается сразу после исправления.
    
- Если объявление отклонено **3 раза подряд** по одной причине, включается “пауза модерации” на 12 часов (для снижения спама).
    

## 5. Автоматическая и ручная модерация

- Автоматическая модерация проверяет текст, изображения, паттерны мошенничества.
    
- Ручная модерация применяется при спорных случаях, жалобах, высокорисковых категориях.
    

## 6. Работа с жалобами

- Жалоба пользователя регистрируется и получает статус.
    
- Срок первичной реакции: до 24 часов.
    
- При подтверждении нарушения объявление скрывается, продавцу выдаётся предупреждение или ограничение.
    

## 7. Исключения и спорные случаи

- Коллекционные предметы с исторической символикой рассматриваются индивидуально. Решение принимается по контексту, без “автоматического разрешения”.
    
- Товары “на запчасти” допускаются, если в названии/описании явно указано “нерабочее” и есть фото фактического состояния.
